# פרוטוטייפ באינטרנט — איך להעלות ולעדכן

האתר מוכן להעלאה. אחרי שתעלה פעם אחת תקבל **קישור באינטרנט** לעבודה. בכל שינוי — מעלים שוב את **כל** התיקייה ומקבלים את אותו קישור מעודכן.

---

## אפשרות 1: Netlify Drop (בלי התקנות)

1. היכנס ל־**[app.netlify.com/drop](https://app.netlify.com/drop)**  
   (אם צריך — הרשם חינם עם אימייל או GitHub.)
2. גרור **את כל תיקיית הפרויקט** לתוך האזור "Drag and drop your site output folder here".  
   התיקייה שצריך לגרור: **`c:\Oren\cursor`** (זו שבתוכה יש `index.html`, `media`, `my_web`).
3. Netlify יעלה את האתר וייתן לך קישור, למשל:  
   **`https://שם-אקראי.netlify.app`**
4. **לעדכן את הפרוטוטייפ:** אחרי ששינית קבצים במחשב — גרור **שוב** את כל תיקיית `c:\Oren\cursor` לאותו דף Drop. האתר יתעדכן לאותו קישור.

---

## אפשרות 2: Surge (מהטרמינל, אם יש לך Node)

אם מותקן אצלך Node.js, אפשר להריץ:

```bash
cd c:\Oren\cursor
npx surge . --domain oren-portfolio-prototype.surge.sh
```

בפעם הראשונה Surge יבקש הרשמה (אימייל + סיסמה). אחרי ההעלאה תקבל קישור כמו:  
**`https://oren-portfolio-prototype.surge.sh`**

לעדכון — הרץ שוב את אותה פקודה מתוך התיקייה.
